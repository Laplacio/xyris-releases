# Xyris Desktop

Xyris is a professional SVG animation editor for creating and editing animated SVG graphics.

**Website**: [https://xyris.app](https://xyris.app)

## Download

Download the latest version from the [Releases](../../releases/latest) page.

| Platform | Download |
|----------|----------|
| Windows (x64) | [Xyris-Desktop-v{version}_setup.exe](../../releases/latest) |

## Getting Started

### Installation
1. Download the `.exe` installer from the latest release
2. Run the installer and follow the setup wizard
3. Launch Xyris from the desktop shortcut or Start Menu
4. Sign in with your Xyris account

### Subscription

A valid subscription is required to use Xyris Desktop. Visit [xyris.app](https://xyris.app) to view pricing and plans.

## Auto-Updates

Xyris Desktop checks for updates automatically on launch. When a new version is available, you will be prompted to download and install it.

## System Requirements

- **OS**: Windows 10 or later (x64)
- **RAM**: 4 GB minimum, 8 GB recommended
- **Disk**: 500 MB free space
